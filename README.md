###Examples:
```
# without input file path
bash$ ./ws.rb
(1,2) (1,0)
(0,2) (0,1)
(2,0) (1,0)
(0,1) (1,0)
NOT FOUND
# with input file path
bash$ ./ws.rb input.txt
(1,2) (1,0)
NOT FOUND
NOT FOUND
NOT FOUND
NOT FOUND
```
